# Where's the Water? (WTW)

For new students, and visiotrs, staying hydrated on campus can be a struggle. Even for continuing students, finding water fountins close to them can be hard to do on a continuing changing campus. Where's the Water? (WTW) aims to solve that by showing water fountains on campus visually visitors can locate water fountains across the University of Reading Campus. With an interactive map and detailed information for each fountain, staying hydrated has never been easier.

## Technologies

This project is built with:

- `HTML`
- `CSS`
- `JavaScript`
- `Leafet.js`

I also used [IconScout](https://iconscout.com/) for the iconography in the website and [Figma](https://www.figma.com/) for intial designing. 

## Features

Users can:
* Zoom in and out of the map that starts in the whitenights campus.
* Hover over bblue markers to reveal the building name
* Select marks that list the water fountain avaliables in the uilding.
* View details for each water fountin like floor level, nearest room and  If the fountain is built into the wall (wall), plugged in dispenser or in a bathroom.

## Future Features

This list covers featured that will potientally be added to imporove the functionality of Where's The Water.

Users should in the future be able to:
* See their location and name the closest building too them
* Filter fountsin by floor level or type
* Flag broken fountain and be able to see flagged fountains on the map
* Be able to see photos of the fountains
* Recommend missing fountains to the system
