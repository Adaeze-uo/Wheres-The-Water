# Where's the Water? (WTW)

A user-friendly web application designed to help students and visitors locate water fountains across the University of Reading Campus. With an interactive map and detailed information for each fountain, staying hydrated has never been easier.

## Features

* **Interactive Campus Map:** A clean, easy-to-navigate map of the University of Reading Whiteknights.
* **Comprehensive Fountain Database:** Find water fountains in libraries, academic buildings, and social spaces.
* **Detailed Fountain Information:** See specific details for each location, such as:
  * Building name and floor level.
  * If the fountain is bulit into the wall (wall), plugged in dispenser or in a bathroom.

## Tech Stack

This project is built with:

* **Frontend:** HTML, CSS, JavaScript
* **Mapping Library:** [Leaflet.js](https://leafletjs.com/) 
* **Icons:** [IconScout](https://iconscout.com/)
